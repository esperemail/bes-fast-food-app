# Bes Fast Food – Web App de Commande

Application de commande en ligne pour fast food, développée avec :

- Google Apps Script
- Google Sheets (backend)
- HTML / CSS / JS (frontend)

## Fonctionnalités
- Menu dynamique depuis Google Sheets
- Panier client
- Commande en ligne
- Adresse obligatoire pour la livraison
- Design mobile-first

## Déploiement
L'application est déployée via Google Apps Script (Web App).
